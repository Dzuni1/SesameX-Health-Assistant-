# CONTRIBUTING

Thanks for your interest in contributing to SesameX Health Assistant!

If you want to contribute:
1. Fork the repository.
2. Create a new branch: `git checkout -b feature/your-feature`.
3. Make changes and commit with clear messages: `git commit -m "Add ..."`.
4. Push to your fork and create a Pull Request describing the change.

Please do not add real API keys in commits. Use `.env` and never commit secrets.
